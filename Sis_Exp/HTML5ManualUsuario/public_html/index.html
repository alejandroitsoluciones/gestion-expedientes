Manual De Usuario

Introducción

Este documento contiene el manual de funcionamiento del sistema de gestión de expedientes, utilizado en el concejo deliberante de la provincia de salta. El sistema de gestión de expediente nos permitirá crear el registro de un expediente que haya entrado al concejo, incluyendo en este los datos de su o sus iniciantes, mover el expediente entre las distintas comisiones del concejo. De esta manera se podrá llevar un control casi preciso de donde se encuentra cada expediente. Para poder trabajar con este sistema deberemos contar con una cuenta de usuario la cual no será suministrada al ingresar al concejo deliberante, estas cuentas estarán relacionadas con la comisión a la cual pertenezcamos ya que varios de las funciones del sistema son exclusivas para la comisión de mesa de entrada. Dentro del sistema existen dos tipos de usuarios, el usuario estándar tendrá acceso solamente ciertas funciones dependiendo de su comisión, así como también solamente podrá trabajar con los expedientes asociados a dicha comisión. El usuario de tipo administrador tendrá acceso a todas las funciones del sistema así como también funciones exclusivas para este tipo de usuario, también este usuario podrá realizar operaciones con todos los expediente del sistema sin importar a la comisión a la cual pertenezca.
